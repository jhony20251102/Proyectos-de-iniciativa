======================================================
        INSTRUCCIONES DE EJECUCIÓN DEL SISTEMA
======================================================

Este sistema está diseñado para ejecutarse localmente. 
Para su correcto funcionamiento, se requiere la instalación 
de XAMPP en el equipo que actuará como servidor.

PASOS PARA LA EJECUCIÓN Y PROYECCIÓN:

1. Instalar XAMPP en el equipo principal.
2. Abrir el Panel de Control de XAMPP e iniciar el servicio de "Apache".
3. Alojar los archivos de este proyecto web dentro de la carpeta 'htdocs' 
   de XAMPP (por lo general ubicada en C:\xampp\htdocs\).
4. Obtener la dirección IP local del equipo principal (por ejemplo, 192.168.1.X).
5. Para proyectar en los televisores (Smart TVs o dispositivos conectados), 
   abrir el navegador web de cada televisor.
6. Ingresar en el navegador la dirección IP local del equipo junto con 
   la ruta del proyecto. Por ejemplo: http://192.168.1.X/tv

De esta manera, los televisores accederán al servidor local (XAMPP) y 
mostrarán el contenido de la web de manera sincronizada.
